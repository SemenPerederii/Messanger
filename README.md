# Messanger

Messanger is een eenvoudige chatapplicatie ontwikkeld als oefening in **Angular** en **Hubs**. Dit project is voornamelijk bedoeld voor leerdoeleinden, en niet alle code is door mijzelf geschreven.

## Functies

- Bekijk welke gebruikers online zijn.  
- Zie wanneer je gesprekspartner een bericht aan het typen is (om dit te testen kun je twee browsers openen).

## Vereisten

- [Docker](https://www.docker.com/) geïnstalleerd op je computer.

## Installatie en starten

1. **Clone de repository:**  
  git clone <repository-url>

2. **Ga naar de projectmap:**
  cd <project-map>

3. **Start het project met Docker Compose:**
  docker compose up --build

  4. **Ga naar** http://localhost:4200

Het project wordt nu gebouwd en gestart. Je kunt het openen in je browser en de functionaliteiten testen.
